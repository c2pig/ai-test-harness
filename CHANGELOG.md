## [1.0.0] - 2025-11-30

### Added
- Core testing framework (`@ai-test-harness/core`)
- CLI tool for scaffolding and running tests (`@ai-test-harness/cli`)
- Support for text-generation and agent-scenario test types
- AWS Bedrock integration
- Multi-tenant testing framework
- Configuration validation with Zod schemas
- Quality assessment framework
- Data connectors (CloudWatch, DynamoDB, PostgreSQL)
